# Release v0.1.0

## Features

- **Agency Orchestration**: Intelligent task management and coordination between multiple AI agents.
- **3D Scene Manager**: Dynamic handling of characters, props, and environment state using Three.js.
- **Character Controller**: Realistic movement, pathfinding, and state machine-based animations.
- **LLM Integration**: Support for Gemini via a unified factory pattern.
- **Interactive UI**: Real-time project overview, kanban board, and agent task monitoring.
- **Dual Logging System**: User-friendly activity log for general agent actions and a technical inspector for detailed LLM traces and responses.
- **Debug Mode**: Ability to pause the simulation on every LLM call for careful inspection.
- **Agent Set Picker**: Capability to switch between different specialized teams of agents.
- **BYOK Support**: Bring Your Own Key support for LLM configuration.

## Technical Details

- **World Environment**: Fixed office scene (`office.glb`) with embedded Points of Interest (POI) for agent interaction.
- Built with React, Three.js, Zustand, TailwindCSS, TypeScript, and Vite.
- Physics-based navigation using three-pathfinding (NavMesh).
- State-driven character expressions and behaviors modeled & rigged with Blender.
- Real-time action logging and inspector panels.
