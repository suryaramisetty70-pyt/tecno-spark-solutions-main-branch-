## 📦 INSTALLATION STEPS

### Step 1: Clone/Download
```bash
# Extract the provided zip file
unzip buddy-ai-os-enhanced.zip
cd buddy-ai-os
```

### Step 2: Install Dependencies
```bash
cd backend
pip install -r requirements.txt
```

### Step 3: Install Free Tools
```bash
# Image generation
pip install diffusers torch transformers

# Content creation
pip install ollama huggingface_hub

# Video processing
pip install opencv-python ffmpeg-python

# Audio
pip install librosa soundfile pyttsx3
```

### Step 4: Start Services
```bash
# Backend
python3 -m api.main

# Frontend (separate terminal)
cd ../frontend-web
npm install
npm start

# Desktop (separate terminal)
cd ../frontend-desktop
npm install
npm start
```
